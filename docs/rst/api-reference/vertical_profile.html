Vertical Profile¶

The vertical profile module contains structures to create vertical profiles for use in DISORT.

class vertical_profile.Conrath(altitude, q0, scale_height, nu)¶

A structure to compute a Conrath profile(s).

Conrath creates Conrath profile(s) on an input grid of altitudes given a volumetric mixing ratio and Conrath parameters. The Conrath profile is defined as

\[q(z) = q_0 * e^{\nu(1 - e^{z/H})}\]

where \(q\) is a volumetric mixing ratio, \(z\) is the altitude, \(\nu\) is the Conrath nu parameter, and \(H\) is the scale height.

__init__(altitude, q0, scale_height, nu)¶
Parameters

  • altitude (ndarray) – The altitude at which to construct a Conrath profile (probably the midpoint altitude)

  • q0 (ndarray) – The surface mixing ratio for each of the Conrath profiles.

  • scale_height (ndarray) – The scale height of each of the Conrath profiles.

  • nu (ndarray) – The nu parameter of each of the Conrath profiles.

Raises

  • TypeError – Raised if any of the inputs are not a numpy.ndarray.

  • ValueError – Raised if any of the inputs do not have the same shape, or if they contain physically or mathematically unrealistic values.

Notes

For an MxN array of pixels, q0, scale_height, and nu should be of shape MxN. altitude should have shape ZxMxN, where Z is the number of altitudes. Additionally, the units of altitude and scale_height should be the same.

property profile¶

Get the Conrath profile(s). This will have the same shape as altitude.

Return type

ndarray

class vertical_profile.Uniform(altitude, bottom, top)¶

A structure to create a uniform profile(s).

Uniform creates uniform volumetric mixing ratio profile(s) on an input grid of altitudes given a set of top and bottom altitudes.

__init__(altitude, bottom, top)¶
Parameters

  • altitude (ndarray) – The altitude boundaries at which to construct uniform profile(s). These are assumed to be decreasing to keep with DISORT’s convention.

  • bottom (ndarray) – The bottom altitudes of each of the profiles.

  • top (ndarray) – The top altitudes of each of the profiles.

Get the uniform profile(s). This will have the same shape as altitude.
