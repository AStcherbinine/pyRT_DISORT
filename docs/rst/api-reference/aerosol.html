Aerosol¶

The aerosol module contains structures to make the aerosol properties required by DISORT.

class aerosol.Conrath(altitude, q0, scale_height, nu)¶

A structure to compute a Conrath profile(s).

Conrath creates Conrath profile(s) on an input grid of altitudes given a volumetric mixing ratio and Conrath parameters.

__init__(altitude, q0, scale_height, nu)¶
Parameters

  • altitude (ndarray) – The altitude at which to construct a Conrath profile (probably the midpoint altitude)

  • q0 (ndarray) – The surface mixing ratio for each of the Conrath profiles.

  • scale_height (ndarray) – The scale height of each of the Conrath profiles.

  • nu (ndarray) – The nu parameter of each of the Conrath profiles.

Raises

  • TypeError – Raised if any of the inputs are not a numpy.ndarray.

  • ValueError – Raised if any of the inputs do not have the same shape, or if they contain physically or mathematically unrealistic values.

Notes

The Conrath profile is defined as

\[q(z) = q_0 * e^{\nu(1 - e^{z/H})}\]

where \(q\) is a volumetric mixing ratio, \(z\) is the altitude, \(\nu\) is the Conrath nu parameter, and \(H\) is the scale height.

For an MxN array of pixels, q0, scale_height, and nu should be of shape MxN. altitude should have shape ZxMxN, where Z is the number of altitudes. Additionally, the units of altitude and scale_height should be the same.

property profile¶

Get the Conrath profile(s). This will have the same shape as altitude.

Return type

ndarray

class aerosol.Interpolator(coefficient, particle_size_grid, wavelength_grid, particle_size, wavelength)¶

A structure to get values over a grid.

Interpolator accepts an array of coefficients, along with the 1D particle size and wavelength grid over which they’re defined. It contains methods to interpolate onto a new grid.

__init__(coefficient, particle_size_grid, wavelength_grid, particle_size, wavelength)¶
Parameters

  • coefficient (ndarray) – ND array of coefficients to interpolate over. axis=-1 is assumed to be the wavelength dimension and axis=-2 is assumed to be the particle size dimension.

  • particle_size_grid (ndarray) – 1D array of particle sizes over which the above properties are defined.

  • wavelength_grid (ndarray) – 1D array of wavelengths over which the above properties are defined.

  • particle_size (ndarray) – 1D array of particle sizes to regrid the properties onto.

  • wavelength (ndarray) – 1D array of wavelengths to regrid the properties onto.

nearest_neighbor()¶

Get the input coefficients at the input particle sizes and wavelengths using nearest neighbor interpolation to the input particle size and wavelength grid.

Return type

ndarray

class aerosol.NearestNeighborTabularLegendreCoefficients(coefficients, particle_size_grid, wavelength_grid, particle_size_profile, wavelengths, max_moments=None)¶

Create a grid of Legendre coefficients (particle size, wavelength).

NearestNeighborTabularLegendreCoefficients accepts a 3D array of Legendre polynomial coefficients of the phase function that depend on particle size and wavelength and casts them to an array of the proper shape for use in DISORT given a vertical particle size gradient.

__init__(coefficients, particle_size_grid, wavelength_grid, particle_size_profile, wavelengths, max_moments=None)¶
Parameters

  • coefficients (ndarray) – 3D array of Legendre coefficients that depend on particle size and wavelength. It is assumed to have shape [n_moments, particle_size_grid, wavelength_grid]

  • particle_size_grid (ndarray) – 1D array of particle sizes associated with the coefficients matrix.

  • wavelength_grid (ndarray) – 1D array of wavelengths associated with the coefficients matrix.

  • wavelengths (ndarray) – ND array of wavelengths where to cast coefficients to.

  • particle_size_profile (ndarray) – 1D array of particle sizes.

  • max_moments (Optional[int]) – The maximum number of coefficients to use in the array. If None, all available coefficients are used.

property phase_function¶

Get the Legendre coefficients cast to the proper grid.

Returns

The Legendre coefficients on a grid.

Return type

np.ndarray

Notes

In DISORT, this variable is named “PMOM”. In addition, The shape of this array is determined by the inputs. In general, the shape will be [n_moments, n_layers, (n_pixels)]. For example, if coefficients has shape [65, 10, 300], altitude_grid has shape [20], and wavelengths has shape [50, 100, 45], this array will have shape [65, 20, 50, 100, 45].

Note that DISORT expects an array of shape [n_moments, n_layers] as the PMOM input, so be sure to select the correct pixel when iterating over pixels.

class aerosol.OpticalDepth(mixing_ratio_profile, column_density_layers, extinction, column_integrated_optical_depth)¶

A structure to compute the optical depth given profiles.

OpticalDepth accepts a mixing ratio profile and atmospheric column density profile to compute the optical depth at each wavelength. It also accepts an extinction profile to scale the optical depth to a reference wavelength, and ensures that the sum over the layers is equal to the total column integrated optical depth.

__init__(mixing_ratio_profile, column_density_layers, extinction, column_integrated_optical_depth)¶
Parameters

  • mixing_ratio_profile (ndarray) – 1D array of mixing ratio.

  • column_density_layers (ndarray) – 1D array of the column density in the layers.

  • extinction (ndarray) – 2D array of extinction coefficients.

  • column_integrated_optical_depth (float) – The total column integrated optical depth.

property total¶

Get the total optical depth in each of the layers.

Return type

ndarray

class aerosol.Uniform(altitude, bottom, top)¶

A structure to create a uniform profile(s).

Uniform creates uniform volumetric mixing ratio profile(s) on an input grid of altitudes given a set of top and bottom altitudes.

__init__(altitude, bottom, top)¶
Parameters

  • altitude (ndarray) – The altitude boundaries at which to construct uniform profile(s). These are assumed to be decreasing to keep with DISORT’s convention.

  • bottom (ndarray) – The bottom altitudes of each of the profiles.

  • top (ndarray) – The top altitudes of each of the profiles.
