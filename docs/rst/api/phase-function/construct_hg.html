pyrt.construct_hg(asymmetry_parameter: ArrayLike, scattering_angles: ArrayLike) → numpy.ndarray[source]Â¶

Construct a Henyey-Greenstein phase function from asymmetry parameters.

Parameters

  • asymmetry_parameter (ArrayLike) â N-dimensional array of asymmetry paramters. All values must be between -1 and 1.

  • scattering_angles (ArrayLike) â 1-dimensional array of scattering angles [degrees].

Returns

N-dimensional arrray of phase functions. This array has a shape of scattering_angles.shape + asymmetry_parameter.shape.

Return type

np.ndarray

Notes

The Henyey-Greenstein phase function is defined as

\[p(\theta) = \frac{1}{4\pi} \frac{1 - g^2} {[1 + g^2 - 2g \cos(\theta)]^\frac{3}{2}}\]

where \(p\) is the phase function, \(\theta\) is the scattering angle, and \(g\) is the asymemtry parameter.

Warning

The normalization for the Henyey-Greenstein phase function is not the same as for a regular phase function. For this phase function,

\[\int_{4\pi} p(\theta) = 1\]

not 4 \(\pi\)! To normalize it, either call normalize() or simply multiply the output by 4 \(\pi\).

Examples

Construct phase functions having 181 scattering angles from an array of asymmetry parameters.

>>> from pathlib import Path
>>> import numpy as np
>>> import pyrt
>>> dust_dir = Path(__file__).parent.parent / 'anc' / 'mars_dust'
>>> g = np.load(dust_dir / 'asymmetry_parameter.npy')
>>> g.shape
(24, 317)
>>> ang = np.load(dust_dir / 'scattering_angles.npy')
>>> ang.shape
(181,)
>>> pyrt.construct_hg(g, ang).shape
(181, 24, 317)
