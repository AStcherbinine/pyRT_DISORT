API Reference¶

Angles¶

azimuth(incidence, emission, phase)

Construct azimuth angles from a set of incidence, emission, and phase angles.

Spectral¶

wavenumber(wavelength)

Convert wavelengths [microns] to wavenumber [1 / cm].

Generic phase function¶

resample(phase_function, scattering_angles, ...)

Resample a phase function to an input a number of points.

normalize(phase_function, scattering_angles)

Normalize an array of phase functions.

decompose(phase_function, scattering_angles, ...)

Decompose a phase function into Legendre coefficients.

fit_asymmetry_parameter(phase_function, ...)

Fit asymmetry parameters to an array of phase functions.

Henyey-Greenstein¶

construct_hg(asymmetry_parameter, ...)

Construct Henyey-Greenstein phase functions from asymmetry parameters.

decompose_hg(asymmetry_parameter, n_moments)

Decompose Henyey-Greenstein phase functions into Legendre coefficients.

Legendre coefficients¶

set_negative_coefficients_to_0(coefficients)

Set an array of Legendre coefficients to 0 after the first coefficient is negative.

reconstruct(coefficients, scattering_angles)

Reconstruct phase functions from an array of Legendre coefficients.
