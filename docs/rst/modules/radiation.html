Radiation¶

The flux module contains data structures for holding the flux variables used in DISORT.

class radiation.IncidentFlux(beam_flux=3.141592653589793, isotropic_flux=0.0)¶

A data structure for holding the incident fluxes.

IncidentFlux creates scalars for the incident beam and isotropic fluxes and performs checks that the input fluxes are valid inputs to DISORT.

__init__(beam_flux=3.141592653589793, isotropic_flux=0.0)¶
Parameters

  • beam_flux (float) – The intensity of the incident beam at the top boundary. Ensure this variable and isotropic_flux have the same units. Note that this is an infinitely wide beam. Default is pi.

  • isotropic_flux (float) – The intensity of the incident beam at the top boundary. Ensure this variable and beam_flux have the same units. Default is 0.0.

Raises

TypeError – Raised if either input flux cannot be cast into a float.

Notes

If thermal_emission==True (defined in ThermalEmission) this is assumed to have the same units as PLKAVG (which defaults to [\(\frac{\text{W}}{\text{m}^2}\)]) and the corresponding incident flux is umu0 * beam_flux and pi * isotropic_flux. If thermal_emission==False, beam_flux and isotropic_flux have arbitrary units, and the output fluxes and intensities are assumed to have the same units as these variables.

property beam_flux¶

Get the input flux of the incident beam at the top boundary.

Notes

In DISORT, this variable is named FBEAM.

Return type

float

property isotropic_flux¶

Get the input flux of isotropic sources at the top boundary.

Notes

In DISORT, this variable is named FISOT.

Return type

float

class radiation.ThermalEmission(thermal_emission=False, bottom_temperature=0.0, top_temperature=0.0, top_emissivity=1.0)¶

A data structure for holding thermal emission variables.

ThermalEmission creates variables needs to include thermal emission in DISORT.

__init__(thermal_emission=False, bottom_temperature=0.0, top_temperature=0.0, top_emissivity=1.0)¶
Parameters

  • thermal_emission (bool) –

    Denote whether to use thermal emission. If True, DISORT will include thermal emission and will need the following variables:

    • bottom_temperature (from this class)

    • top_temperature (from this class)

    • top_emissivity (from this class)

    • low_wavenumber (from observation.Spectral)

    • high_wavenumber (from observation.Spectral)

    • temperature (from class:eos.Hydrostatic)

    If False, DISORT will save computation time by ignoring all thermal emission and all of the aforementioned variables. Default is False.

  • bottom_temperature (float) – The temperature of the bottom boundary [K]. Only used by DISORT if thermal_emission==True. Default is 0.0.

  • top_temperature (float) – The temperature of the top boundary [K]. Only used by DISORT if thermal_emission==True. Default is 0.0.

  • top_emissivity (float) – The emissivity of the top boundary. Only used by DISORT if thermal_emission==True. Default is 1.0.

Raises

  • TypeError – Raised if bottom_temperature, top_temperature, or top_emissivity cannot be cast to a float.

  • ValueError – Raised if bottom_temperature or top_temperature is negative, or if top_emissivity is not between 0 and 1.
