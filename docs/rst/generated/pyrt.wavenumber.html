pyrt.wavenumber¶

pyrt.wavenumber(wavelength: ArrayLike) → numpy.ndarray[source]¶

Convert wavelengths [microns] to wavenumber [\(\frac{1}{\text{cm}}\)].

Parameters

wavelength (ArrayLike) – N-dimensional array of wavelengths. All values must be between 0.1 and 50 microns (I assume this is the valid range to do radiative transfer).

Returns

N-dimensional array of wavenumbers with shape wavelength.shape.

Return type

np.ndarray

Raises

  • TypeError – Raised if the input cannot be cast into an ndarray.

  • ValueError – Raised if the input contains values outside the valid range.

Examples

Convert wavelengths to wavenumbers.

>>> import numpy as np
>>> import pyrt
>>> wavs = [1, 2, 3]
>>> pyrt.wavenumber(wavs)
array([10000.        ,  5000.        ,  3333.33333333])

This function can handle arrays of any shape.

>>> wavs = np.ones((10, 20, 30))
>>> pyrt.wavenumber(wavs).shape
(10, 20, 30)
