Contributions
=============

Found a bug? Know of an algorithm that's faster and/or more accurate than what
I use? If so, please let me know by `submitting an issue
<https://github.com/kconnour/pyRT_DISORT/issues>`_. You're welcome to submit a
pull request as well, though I reckon it would be easiest to just ask me to do
it.

If you'd like a new feature, feel free to `start a discussion
<https://github.com/kconnour/pyRT_DISORT/discussions>`_.
