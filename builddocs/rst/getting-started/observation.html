Wavelengths

pyRT_DISORT attempts to keep the structure of your data wherever possible. Let's consider the most generalized case: a hyperspectral imager. If the image is MxN pixels with W wavelengths, the image will have shape MxNxW. I'll go ahead and make a fake spectral array. For simplicity, I'll assume each pixel has the same wavelengths, although that's not a necessity.

Note

The wavelengths must be in microns.

I've defined wavelengths at 1, 2, 3, 4, and 5 microns, and the spectral width of each bin is 0.1 microns. pyRT_DISORT's Wavelengths class holds these values and the associated wavenumbers of each pixel. These can be accessed via the class properties at any time.
